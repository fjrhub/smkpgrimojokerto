# SMK PGRI Mojokerto Website

This project is a school website developed using **Next.js** and **MongoDB** as the database, and deployed on **Vercel**.

The website includes an **announcement management system** that allows administrators to create announcements with a **title, image, and content**. All announcements are stored in MongoDB and displayed dynamically on the website.

## Features
- School profile and information pages
- Announcement system (CRUD)
- Image upload for announcements
- Dynamic content stored in MongoDB
- Responsive design
- Deployed on Vercel

## Tech Stack
- Next.js
- MongoDB
- Vercel

## Purpose
This project was developed as a **PKK (Produk Kreatif dan Kewirausahaan)** project for the **Computer and Network Engineering (TKJ)** major.
